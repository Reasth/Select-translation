为什么顺手

小到一个圆点，
但每个细节都为"不打断你"而设计。

它不抢你的剪贴板、不占你的窗口，只在你需要的那一秒出现。

全局划词

浏览器、PDF 阅读器、IDE、记事本、聊天软件——任何应用里拖选都能触发。

流式译文

token 实时追加渲染，第一个字几乎瞬间出现，不用盯着转圈干等。

跟随光标的圆点

有道词典 / Bob 那种顺手：圆点贴着选区出现，约 2 秒自动隐去，不点就走。

任意大模型

OpenAI 兼容协议，换服务商只改 Base URL 和模型名。默认 MiniMax-M2.1。

Windows + macOS

同一套代码、同一种体验。平台差异只隔离在一个文件里，行为完全对齐。

常驻托盘

菜单栏 / 系统托盘一键暂停、设置、退出。配置存在本地 ~/.translate-popup。

三步

从选中到译文，
一次顺滑的手势。

01

选中

拖选任意一段文本。单击不会触发——避免误判，也不打断你正常复制。

02

出现「译」

光标右上角浮起蓝色小圆点，大约 2 秒后自动隐去；移上去它会留下来等你。

03

点击翻译

弹出卡片，流式渲染译文。Esc 关闭，也能一键复制结果。

你的模型，你做主

插上任意 OpenAI-compatible 服务就能跑。

MiniMax-M2.1 DeepSeek 通义千问 · DashScope 智谱 GLM Moonshot Kimi OpenAI Ollama 本地

中英自动反向：选中中文译成英文，选中英文译成中文。预设里切换服务商会自动填好 Base URL 和模型名，填上 API Key 即用。

安装

几行命令，装到你的电脑。

需要 Python 3。克隆仓库、装依赖、跑起来——就这样。

$ git clone https://github.com/horton2048/Select-translation.git
$ cd Select-translation
$ pip install -r requirements.txt
$ python main.py
